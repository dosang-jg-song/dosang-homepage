SAP DRC는 글로벌 세무 컴플라이언스의 표준 솔루션이지만, 모든 조직에 적합하지는 않습니다. 본 체크리스트는 도입 적합성을 빠르게 판단할 수 있게 합니다.

## 1. 사전 요건 (5문항)

- [ ] SAP S/4HANA 1909 이상 운영 중
- [ ] SAP NW 7.50+ 또는 RISE 환경
- [ ] SAP Cloud Connector 운영 가능
- [ ] 글로벌 마스터 데이터 거버넌스 체계 존재
- [ ] 세무팀의 SAP 직접 사용 경험

## 2. 적용 국가 (5문항)

- [ ] 5개국 이상에서 SAP 운영
- [ ] 운영 중인 국가가 DRC 지원 시나리오에 포함
- [ ] 신규 진출 예정 국가가 1년 내 3개국 이상
- [ ] 기존 자체 솔루션의 유지보수 부담
- [ ] 연 거래량 100K+ 인보이스

## 3. 거버넌스 (5문항)

- [ ] 세무·IT 양 부서의 공동 추진 체계
- [ ] CFO 직속 또는 임원급 의사결정 가능
- [ ] 외부 감사인의 시스템 검증 요구
- [ ] 분기 단위 컴플라이언스 보고 의무
- [ ] 사고 시 법인장 책임 명시

## 4. 현 운영 (5문항)

- [ ] 자체 솔루션의 신뢰도 이슈 발생
- [ ] 신규 국가 추가 시 6개월+ 소요
- [ ] eInvoice 의무화 국가 5개국 이상
- [ ] Tax Compliance 인력 5명 미만
- [ ] 외부 컨설팅 의존도 70%+

## 5. 비용 (5문항)

- [ ] 현 자체 솔루션 연 운영비 €500K+
- [ ] DRC 라이선스 예산 확보 가능
- [ ] 도입 프로젝트 예산 €1M+ 가능
- [ ] 5년 TCO 비교 가능
- [ ] 단계적 롤아웃 가능 (12~24개월)

## 결과 해석

- **20개 이상 체크**: DRC 도입 강력 권고
- **15~19개**: 단계적 도입 검토
- **10~14개**: 우선 일부 국가 PoC
- **9개 이하**: 자체 솔루션 또는 Tax Engine 검토
